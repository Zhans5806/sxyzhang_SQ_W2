# Week 2 Side Quest: Platformer with Platforms Array

## What This Example Demonstrates
This example demostrates a sushi roll jumping across platforms in a sushi store. 

## Setup and Interaction Instructions

To run the code open it in vscode and run through index.html

**Controls:**
Controls are either with the arrow keys or WASD
